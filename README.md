# ANDCODEDIT

**A powerful VS Code-like code editor for Android**

## ✨ Core Features

- **Full-featured Code Editor** like VSCode on Android
- **DEX Mode** - View, edit, decompile and patch DEX/ APK files directly
- **Android 16 Native Desktop Mode** - True desktop experience with resizable panels, multi-window support
- **Full Linux Terminal** - Direct access to Android's underlying Linux shell with PTY support

- Rust (for high-performance terminal backend)
- dexlib2 / smali for DEX manipulation
- Android 16 Desktop APIs
