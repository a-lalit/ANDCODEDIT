# ANDCODEDIT

A powerful Android code editor with DEX mode, Android 16 native desktop mode, and full Linux shell terminal support.

## Features
- VS Code-like editor experience on Android
- Full interactive Linux terminal
- DEX mode support
- Adaptive UI for mobile + DeX + Desktop mode
- Language support (Kotlin, Java, Rust, C/C++)

Built with Kotlin, Jetpack Compose, Rust (PTY), and Material 3.
